# PIK Exchange Bot

رباتِ تلگرامیِ صرافیِ کوچک برای خرید/فروش **روبل** و **دلار**، با نرخ لحظه‌ای از
[alanchand.com](https://alanchand.com/en)، پنل مدیریت داخل تلگرام، و نصب/آپدیت یک‌خطی
روی سرور (مثل ربات اطلس).

## امکانات

- منوی خرید روبل تینکوف با **پله‌های حجمی** و ضریب جداگانه‌ی هر پله.
- خرید **روبل تک‌فیشی**، **دلار نقدی**، **فروش دلار مهردار/کهنه**، **خرید تومان با روبل**.
- نرخ‌ها یا **فرمولی** (نرخ سایت × ضریب) هستند یا **دستی** (مقدار ثابت ادمین).
- نمایش **قوانین معاملات**، دکمه‌ی موافقت و **دریافت کارت**، سپس **ارسال فیش** و تأیید توسط ادمین.
- **کیف پول** و **معاملات من** برای کاربر.
- **پنل مدیریت داخل تلگرام**: تنظیم ضریب‌ها و پله‌ها، شماره کارت، نرخ دلار، متن قوانین/خوش‌آمد/پشتیبانی،
  مشاهده و تأیید تراکنش‌ها، شارژ کیف پول، پیام همگانی، نرخ لحظه‌ای سایت.
- **اطلاع‌رسانی آپدیت**: بعد از هر آپدیت، به ادمین خبر می‌دهد و با یک دکمه می‌توان به همه‌ی کاربران اطلاع داد.
- دستور مدیریتی ترمینال: `pik` (شبیه `x-ui`).

## پیش‌نیاز: تنظیم آدرس ریپازیتوری

قبل از نصب یک‌خطی، فایل [`bootstrap.sh`](bootstrap.sh) را ویرایش کنید و مقدار `REPO_URL`
را با آدرس ریپازیتوری گیت‌هاب خودتان جایگزین کنید (یا هنگام اجرا `REPO_URL=...` بدهید):

```bash
REPO_URL="https://github.com/majazi041999-spec/PIKExchange.git"
```

## نصب یک‌خطی

```bash
bash <(curl -Ls https://raw.githubusercontent.com/majazi041999-spec/PIKExchange/main/bootstrap.sh)
```

این کار:
1. ریپو را در `/opt/PIKExchangeBot` کلون/به‌روزرسانی می‌کند.
2. `install.sh` را اجرا و توکن/ادمین/پشتیبانی را می‌پرسد.
3. سرویس `pik-bot` را می‌سازد و اجرا می‌کند.
4. دستور `/usr/local/bin/pik` را نصب می‌کند.

هنگام نصب این‌ها را وارد می‌کنید:

| مقدار | توضیح |
|-------|-------|
| `BOT_TOKEN` | توکن ربات از [@BotFather](https://t.me/BotFather) |
| `ADMIN_IDS` | آیدی عددی ادمین‌ها از [@userinfobot](https://t.me/userinfobot) (با کاما جدا کنید) |
| `SUPPORT_USERNAME` | آیدی/لینک پشتیبانی که به کاربر نشان داده می‌شود |

## آپدیت

```bash
pik update
# یا
bash <(curl -Ls https://raw.githubusercontent.com/majazi041999-spec/PIKExchange/main/bootstrap.sh) update
```

آپدیت به‌صورت امن انجام می‌شود (auto-stash تغییرات محلی، نصب پکیج‌های جدید، ری‌استارت سرویس).

## فقط تنظیم مجدد توکن/ادمین

```bash
pik configure
```

## دستور `pik` (مدیریت ترمینال)

بعد از نصب کافیست بزنید:

```bash
pik
```

منوی تعاملی باز می‌شود. حالت دستوری هم دارد:

```bash
pik status      # وضعیت سرویس
pik logs        # لاگ زنده
pik restart     # ری‌استارت
pik update      # آپدیت امن
pik configure   # تنظیم .env
pik uninstall   # حذف سرویس (داده‌ها می‌مانند)
```

## پنل مدیریت (داخل تلگرام)

ادمین دستور `/admin` را می‌زند یا از منوی اصلی دکمه‌ی «🛠 پنل مدیریت» را انتخاب می‌کند.

- **💱 نرخ‌ها و محصولات** — برای هر محصول: ضریب پله‌ها، سقف‌ها، برچسب، ستون سایت (خرید/فروش/میانگین)،
  حالت دستی/فرمولی و نرخ دستی. پیش‌نمایش نرخ لحظه‌ای هم نمایش داده می‌شود.
- **💳 مدیریت کارت‌ها** — چند کارت واریز ثبت کنید (هر کدام یک برچسب + متن کامل)، و انتخاب کنید ربات
  کدام را نشان دهد. هنگام دریافت کارت، ربات **دو پیام** می‌فرستد: تأیید+نرخ، سپس متن کارت به‌صورت جدا.
- **📝 متن‌ها** — قوانین معاملات، خوش‌آمد، پشتیبانی.
- **📊 تراکنش‌ها** — لیست و تأیید/رد فیش‌ها؛ پس از تأیید، ابزار ارتباط با کاربر (لینک مستقیم یا پیام از طریق ربات).
- **👛 شارژ کیف پول** — افزایش/کاهش موجودی کاربر.
- **📈 نرخ لحظه‌ای سایت** — نرخ خام سایت و نرخ محاسبه‌شده‌ی همه‌ی محصولات.
- **📢 پیام همگانی** — ارسال پیام به همه‌ی کاربران.
- **🎨 دکمه‌های رنگی** — روشن/خاموش کردن رنگ دکمه‌ها با یک دکمه (پیش‌فرض: خاموش).

## فرمول نرخ‌ها (پیش‌فرض)

نرخ روبل از **ستون فروش** سایت به **تومان** تبدیل می‌شود (مقدار ریالی ÷ ۱۰) و سپس:

| محصول / پله | ضریب |
|-------------|------|
| تینکفی — زیر ۲۵ میلیون تومان | ۰٫۹۳ |
| تینکفی — ۲۵ تا ۱۵۰ میلیون تومان | ۰٫۹۲ |
| تینکفی — بالای ۱۵۰ میلیون تومان | ۰٫۹۱۵ |
| تک‌فیش | ۰٫۹۶۵ |

نرخ محصولات ساده = **مبنا × ضریب + آفست**:

| محصول | مبنا | فرمول | واحد نتیجه |
|-------|------|-------|-----------|
| خرید تومان با روبل | نرخ فروش روبل سایت | مبنا − ۳۰۰ | تومان |
| فروش دلار مهردار/کهنه | دلار→روبل بانک روسیه (CBR) | مبنا − ۸ | **روبل** |
| خرید دلار نقدی | دلار سایت | مبنا × ۱ | تومان |

> همه‌ی ضریب‌ها، آفست‌ها و سقف‌ها از پنل تلگرام قابل تغییرند. برای دلار مهردار می‌توانید نرخ
> دقیق اسبر را دستی وارد کنید (اولویت با نرخ دستی است)؛ در غیر این صورت از نرخ رسمی بانک روسیه
> استفاده می‌شود. (سایت خود اسبربانک از خارج روسیه API قابل‌دسترسی ندارد.)

## ساختار پروژه

```
main.py                 # نقطه‌ی ورود؛ راه‌اندازی ربات + ورکر تازه‌سازی نرخ
core/
  config.py             # خواندن .env
  db.py                 # SQLite (کاربران، تراکنش‌ها، تنظیمات، کیف‌پول)
  products.py           # تعریف محصولات و بازنویسی‌های ذخیره‌شده
  rates.py              # دریافت نرخ از سایت + محاسبه‌ی نرخ محصولات
  texts.py              # متن‌های پیش‌فرض (قوانین/خوش‌آمد/پشتیبانی)
  updates.py            # اطلاع‌رسانی آپدیت به ادمین/کاربران
  jalali.py, utils.py   # تاریخ شمسی و ارقام فارسی
bot/
  keyboards.py          # کیبوردهای اینلاین
  states.py             # حالت‌های FSM
  handlers/
    user.py             # منو، نرخ، قوانین، دریافت کارت، ارسال فیش
    admin.py            # پنل مدیریت
install.sh update.sh setup_service.sh bootstrap.sh uninstall.sh pik_menu.sh
```

## اجرای دستی (بدون سرویس، برای تست محلی)

```bash
python3 -m venv .venv
source .venv/bin/activate      # ویندوز: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env           # سپس BOT_TOKEN و ADMIN_IDS را پر کنید
python main.py
```
